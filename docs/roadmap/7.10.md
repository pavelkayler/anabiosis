# 7.10. Long/short ratio soft sentiment feature

## Статус дорожной карты
- Текущий этап: `7`
- Текущий md-файл: `7.10.md`
- Статус файла: `active`
- После полного завершения файл должен быть переименован в: `7.10_done.md`

---

## Цель подэтапа
Добавить long/short ratio как мягкую контекстную sentiment-feature без превращения её в hard gate для торгового решения.

---

## Зависимости
- `5.3.md`
- `7.9.md`
- `8.4.md`
- `8.7.md`

---

## Что должно быть получено по итогам подэтапа
- storage for long/short ratio snapshots
- normalized long/short ratio feature
- integration into confidence/context layer
- UI/debug visibility

---

## Обязательные правила
- long/short ratio не должен в одиночку создавать `ENTER_*`;
- long/short ratio не должен в одиночку invalid-ить setup;
- он используется только как soft feature для context / confidence / warning;
- отсутствие long/short ratio не должно ломать pipeline.

### Где он может влиять
- усиливать squeeze-prone reading;
- усиливать crowding warning;
- ослаблять continuation quality при extreme crowding;
- участвовать в explanation text.

### Где он не должен влиять
- не заменяет structure;
- не заменяет CVD;
- не заменяет OI expansion;
- не заменяет valid setup geometry.
